Cómo usar la API

1. Subir un documento

POST /api/v1/documentos
X-API-Key: tu_clave_aqui
Content-Type: multipart/form-data

file=@factura.pdf

# Respuesta:
{ "ok": true, "job_id": 42 }

2. Consultar el resultado

GET /api/v1/documentos/42
X-API-Key: tu_clave_aqui

# Respuesta cuando estado="completado":
{
  "ok": true,
  "job_id": 42,
  "estado": "completado",
  "tipo_documento": "factura_a",
  "metodo_extraccion": "pdfplumber",
  "cae_valido": true,
  "cae_vencimiento": "2026-06-30",
  "resultado": {
    "cae": "12345678901234",
    "cuit_emisor": "30-99999999-9",
    "importe_total": 15000.00,
    "fecha_emision": "2026-04-25",
    ...
  }
}

3. Ejemplo con curl

curl -X POST https://iafacturas.dewoc.com/api/v1/documentos \
  -H "X-API-Key: tu_clave_aqui" \
  -F "file=@factura.pdf"
